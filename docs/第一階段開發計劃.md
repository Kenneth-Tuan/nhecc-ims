# 第一階段開發計劃

本階段目標為建立系統核心基礎建設，並優先完成裝備與牧養系統，以利教會日常核心運作數位化。

| 功能名稱                | 詳細項目                                                                                                                                         | 預估工時 (人/日) |
| :---------------------- | :----------------------------------------------------------------------------------------------------------------------------------------------- | :--------------: |
| **1. 會友資料中心**     | 1. 基本資料管理（CRUD + 篩選排序）<br>2. 資料匯入/匯出功能                                                                                       |        5         |
| **2. 註冊/登入功能**    | 1. 第三方認證整合（LINE、FB、Google）<br>2. Email / SMS 通知服務串接                                                                             |        5         |
| **3. 系統使用權限管理** | 1. 角色管理（CRUD）<br>2. 角色權限設定與邏輯實作                                                                                                 |        3         |
| **4. 系統操作軌跡**     | 1. 使用者操作行為記錄 (Audit Logs)<br>2. 敏感資料存取監控                                                                                        |        3         |
| **5. 裝備系統**         | 1. 課程管理（CRUD、模糊搜尋、排序）<br>2. 課程點名功能（含 QR Code 掃描）<br>3. 線上報名與請假流程<br>4. 作業繳交與批改機制<br>5. 報表匯出功能   |        13        |
| **6. 牧養系統**         | 1. 牧區/小組組織架構管理<br>2. 牧養/出勤紀錄（含加密、代禱、探訪）<br>3. 屬靈歷程 (Milestones) 紀錄<br>4. 小組管理（行事曆、服事安排、收支紀錄） |        11        |

### 總計預估工時：約 40 人/日

> **備註**：
>
> 1. 工時僅為後端 API 與前端頁面開發及單元測試之粗估，**不包含**：
>    - 詳細需求訪談與確認
>    - UI/UX 設計與完稿
>    - 系統整合測試 (SIT) 與使用者驗收測試 (UAT)
>    - 伺服器部署與環境建置
