## 裝備/牧養系統 (Tranning & shepherding system)

### 概述

本系統旨在協助教會進行全面的會友裝備與牧養關懷，透過數位化工具提升牧養效率並深化肢體連結。

**主要使用情境 (User Stories):**

- **一般會友**：

  - 「作為會友，我希望能透過系統瀏覽並**報名裝備課程**，查看上課時間、地點，並在需要時線上**請假**或**繳交作業**。」
  - 「作為會友，我希望能查看小組的**行事曆**與**服事安排**，清楚知道自己何時需要參與服事。」

- **小組長/牧養同工**：

  - 「作為小組長，我需要一個方便的工具來**記錄小組聚會出席**、**代禱事項**及**探訪紀錄**，以便持續追蹤並關懷組員狀況。」
  - 「作為小組長，我希望能管理小組的**年度規劃**與**收支紀錄**，使小組運作更加透明有序。」

- **牧者/行政管理者**：
  - 「作為管理者，我希望能輕鬆**開設與管理課程**，並透過**報表**掌握整體的裝備與出席狀況。」
  - 「作為牧者，我需要一個**權限分級且資料加密**的環境來查閱牧養紀錄，確保會友的隱私受到保護，同時能掌握牧區的整體情況。」

> REMARK：未來可加入其他事工單位的課程，例如親子蒙氏角落、家庭事工婚姻/交友輔導課程、悠活體適能運動課程…

### 詳細功能

1. **裝備系統 (Training)**
   - 課程管理
     - CRUD；新增/瀏覽/更新/刪除課程
     - 課程模糊搜尋、排序功能
     - 課程點名功能 (支援 QR Code 掃描報到)
     - 課程報名、請假功能
     - 作業繳交/批改（評分）功能
   - 報表匯出
2. **牧養系統 (Shepherding)**
   - 牧區 -> 小組/單位 架構管理
   - 牧養/出勤紀錄 (隱私加密)
     - 代禱事項/RPG/探訪紀錄
     - 小組/禱告會/特會/主日出勤紀錄
     - 歷程 (Milestones) 紀錄（受洗日、裝備完成、服事歷程等）
   - 小組管理
     - 行事曆、年度規劃
     - 小組服事安排表
     - 收支紀錄

## 資源管理系統 (Resource Management System)

### 概述

本系統旨在整合教會各項實體與數位資源，透過統一的平台進行場地、設備、官網內容及莊園作物的管理，優化資源分配流程並提升使用率。

**主要使用情境 (User Stories):**

- **一般會友/同工**：

  - 「作為同工，我希望能線上查詢並**租借場地與設備**，清楚知道可用時段與申請條件，避免檔期衝突。」
  - 「作為會友，我希望能透過官網輕鬆**報名活動**或進行**線上奉獻**，並隨時獲取教會最新資訊。」

- **行政管理者**：

  - 「作為行政人員，我需要一個高效的介面來**審核場地與設備申請**，並透過**自動化檢核**（如資格、時間限制）減少人工溝通成本。」
  - 「作為管理者，我希望能透過**儀表板**即時掌握設備狀態與庫存，並安排**定期維護**，確保資源妥善運作。」
  - 「作為管理者，我希望能匯出**使用頻率報表**，以作為未來資源採購與分配的決策依據。」

- **網站管理者**：

  - 「作為網站管理者，我希望能透過簡單易用的 **CMS** 更新官網內容，無需依賴工程師即可發布消息。」

- **莊園管理者/同工**：
  - 「作為管理者，我希望能透過**儀表板**監控農作物的生長環境數據（如溫濕度），並記錄**生長與收成數據**。」
  - 「作為同工，我希望能清楚知道每日的**農務工作項目**，以便有效率地協助莊園運作。」

### 詳細功能

1. **場地管理**
   - 場地事件行事曆（租借/維護）
   - 場地租借申請條件設定
     - 申請人角色設定
     - 必須提前多久時間申請
   - 創建/瀏覽/更新/刪除場地
   - 申請場地 (支援週期性預約功能)
   - 審核場地申請表單
   - 重複確認場地申請
   - 場地租借之一次性使用 QRCode
   - 復原場地之檢核功能
   - 報表匯出（使用頻率…）
2. **倉儲管理**
   - 設備管理（莊園/文德/碧湖）
     - 設備狀態儀表板
     - 新增/瀏覽/更新/刪除設備
     - 設備維護紀錄&排程
     - 設備異動紀錄
   - 設備租借
3. **NHECC 官網及其後台**
   - 內容管理系統（CMS）
   - 線上奉獻
   - 活動報名
   - 聯絡表單
4. **好牧人莊園作物管理**
   - 作物/環境狀態儀表板
   - 環境監測數據紀錄（溫濕度、光照等）
   - 作物生長日誌（照片/狀態記錄）
   - 農務工作指派與追蹤
   - 收成紀錄與統計

> REMARK：由於 NHECC 和 彩虹全人關懷協會 兩個官方網站屬於不同性質的機構，後續可能會有不同的管理人員以及網站 SEO 的緣故，決定將 彩虹全人關懷協會的官網前後網域獨立出來，但功能和介面可以共用。

## 服事管理系統 (Service System)

### 概述

本系統旨在將全教會的服事資源透明化，提供一個整合性的平台來管理主日服事、禱告會、詩歌資源及各類專案活動。透過歷史資料分析，協助各單位掌握狀態與趨勢，進而優化未來的事工規劃。

**主要使用情境 (User Stories):**

- **服事同工/敬拜團成員**：

  - 「作為敬拜團成員，我希望能線上存取**詩歌庫**，快速搜尋歌譜、進行**轉調**並下載所需的檔案 (PDF/PPT)，以節省備歌時間。」
  - 「作為服事同工，我希望能清楚查看**服事行事曆**與**排班表**，確認自己的服事時間與內容，避免時間衝突。」

- **事工/專案負責人**：

  - 「作為專案負責人，我希望能透過標準化的**SOP 申請流程**來建立新專案，自動連結所需的**場地、設備與經費申請**，減少行政溝通往返。」
  - 「作為活動策劃者，我希望能透過**專案進度儀表板**即時監控各項任務的完成度，確保活動按計畫順利進行。」

- **教會決策者**：
  - 「作為決策者，我希望能透過系統匯出的**歷史資料與報表**，分析各單位的服事量能與趨勢，以作為年度規劃與資源分配的參考依據。」

### 詳細功能

1. **主日服事**
   - 行事曆
2. **禱告會（早/晚/烈火）**
   - 排班表
3. **詩歌管理系統**
   - 詩歌庫管理
     - 新增/瀏覽/更新/刪除詩歌
     - 詩歌標籤功能
     - 歌譜轉調
     - 歌譜 pdf/歌詞 ppt 匯出
4. **專案（服事/活動/短宣）管理**
   - 新增/刪除/瀏覽/更新專案
   - 模糊搜尋
   - 專案申請 template（建立固定的申請流程 SOP，減少人力資源）
     - 連結場地、設備申請
     - 金費申請
   - 專案進度儀表板/行事曆
   - 報表匯出

## 人資系統 (HR System)

### 概述

本系統為獨立運作的行政管理平台，專注於處理高度敏感的人事資料，確保符合法規要求並保障員工隱私。

**主要使用情境 (User Stories):**

- **教會員工**：

  - 「作為員工，我希望能透過手機 App 進行**GPS/Wifi 打卡**，方便記錄出勤時間，即使在外勤也能輕鬆簽到。」
  - 「作為員工，我希望能線上**申請請假**並查看**簽核進度**，以及隨時查閱自己的**薪資單**與**出勤紀錄**。」

- **人資管理者**：
  - 「作為人資，我需要一個安全的環境來管理**員工基本資料**與**薪資結算**，確保資料不外洩。」
  - 「作為人資，我希望能快速**審核請假申請**，並匯出**考勤報表**以利薪資計算。」

> REMARK：視情況可將資料隔離且獨立運作。

### 詳細功能

1. **員工資料管理**

   - 員工基本資料管理 (CRUD)
   - 職位與部門組織架構設定
   - 到職/離職流程管理
   - 員工檔案上傳 (如合約、證件)

2. **考勤管理**

   - GPS/Wifi 打卡功能 (Mobile App/Web)
     - 支援外勤打卡與定位檢核
   - 補打卡申請與審核流程
   - 排班與輪班管理
   - 異常出勤通知與報表

3. **差勤管理 (請假)**

   - 假別設定 (特休、病假、事假、補休等)
   - 特休天數自動計算
   - 線上請假申請與多級簽核
   - 代理人機制
   - 請假紀錄查詢

4. **薪資結算**
   - 薪資結構設定 (本薪、津貼、獎金、勞健保扣除額)
   - 考勤資料自動串接計薪
   - 薪資單自動生成與發送 (PDF/Email 需加密)
   - 薪資報表匯出 (供財務對帳)

## 彩虹全人關懷協會官網

### 概述

本網站作為彩虹全人關懷協會的對外窗口，雖然網域與教會官網獨立，但共享後台功能與介面設計，以維持品牌一致性並降低開發維護成本。

**主要使用情境 (User Stories):**

- **一般民眾/受助者**：
  - 「作為民眾，我希望能透過官網了解協會的**服務項目**與**活動資訊**，並進行**線上捐款**或**志工報名**。」
- **協會管理者**：
  - 「作為管理者，我希望能共用教會的 **CMS 系統**來管理官網內容，無需重新學習一套新系統。」

### 詳細功能

1. **網站前台畫面**

   - 首頁/關於我們/最新消息
   - 服務項目介紹
   - 活動資訊與報名
   - 線上捐款頁面 (串接金流)
   - 志工招募表單

2. **後台系統**
   - 前台內容管理 (CMS)
   - 捐款紀錄查詢與匯出
   - 志工資料管理
   - 聯絡表單回應

> REMARK：參考 NHECC 官網；  
> NHECC 和 彩虹全人關懷協會 兩個官方網站由於屬於不同性質的機構，後續可能會有不同的管理人員以及網站 SEO 的緣故，決定將 彩虹全人關懷協會的官網前後網域獨立出來，但功能和介面可以共用。

## 額外功能

### 概述

本章節包含提升使用者體驗與系統整合性的加值功能，透過與第三方平台 (LINE) 整合及個人化服務，增加系統的便利性。

**主要使用情境 (User Stories):**

- **全體使用者**：
  - 「作為使用者，我希望能透過 **Line 接收系統通知**（如課程提醒、審核結果），即時掌握重要訊息。」
  - 「作為使用者，我希望能擁有**個人化行事曆**，整合所有我有參與的課程、服事與活動，一目了然。」

### 詳細功能

- **Line Message API 與系統功能整合**
- **個人化 Calendar**
- **系統通知功能（Line 訊息、Email、SMS）**
